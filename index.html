My Journey

From novice to ninja, follow my journey as I transform coffee into code and ideas into reality.

Phase 1: Early Exploration (Jan – Mar 2024)

Started learning the basics of web development (HTML, CSS). Built simple static pages to understand structure and styling. Explored UI/UX fundamentals and got introduced to Figma.

Phase 2: Building Foundations (Apr – Jun 2024)

Learned JavaScript for interactivity and dynamic behavior. Practiced responsive web design for mobile and desktop compatibility. Created small projects: landing pages, portfolio sites, and form designs.

Phase 3: Practical Application (Jul – Dec 2024)

Improved Figma skills: wireframing, prototyping, and polished UI designs. Competed in and won 1st place in a Figma design competition. Earned 2nd place in a coding competition for problem-solving and coding speed. Built more advanced projects combining design + code.

Phase 4: Growth & Professional Focus (Jan 2025 – Present)

Continued advancing in frontend development (JS projects, clean code practices). Enhanced skills in UI/UX design for real-world projects. Started working on freelance gigs, portfolios, and professional website designs. Expanding expertise in branding, SEO-ready designs, and client-focused solutions.
